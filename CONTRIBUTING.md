# Contributing

This repository holds decibri's Contributor License Agreement enforcement
tooling. It is published so that decibri repositories can reference the Action,
and it is maintained solely by decibri.

## We do not accept external contributions

This repository does not accept pull requests or external code contributions. It
is operational tooling for decibri's own projects, not a community project. Pull
requests opened here will be closed without review.

If you have found a bug or have a suggestion, please open an issue so a
maintainer can look at it. For anything else, contact the maintainers.

## Contributing to decibri projects

If you meant to contribute to a decibri project that uses this CLA check, do that
in the project's own repository. When you open a pull request there, the CLA
check will tell you whether you are covered and, if not, how to sign.

## Removal requests

To request access to, correction of, or erasure of a CLA signature record, email
privacy@decibri.com. See the [privacy notice](https://decibri.com/privacy) for
full details.
